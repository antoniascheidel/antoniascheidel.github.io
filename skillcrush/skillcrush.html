My Skillcrush Experience So Far

So far, dealing with the code editor has been a mixed experience. Here are some pros and cons, to be taken with a grain of salt because I realize that I'm not the target audience.

Pros

  • It's always nice to see changes in real time, especially with the css.
  • I like the challenge of having to actually type out everything without relying on the autocomplete that Sublime Text does for me.
  • I still love everything about this blueprint, so it'll be a while before the cons eat up all that love. And I realize that the editor is just a place to get started, and it's awesome for that!

Cons

  • Being used to Sublime Text, I do miss being able to move lines around and typing out every tag by hand does get old pretty quickly.
  • The automatic indentation is not consistent and makes organizing the code a bit difficult.
